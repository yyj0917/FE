## 📌 서비스 소개

Runway는 러닝 코스와 주변 관광지를 함께 추천하는 서비스입니다.

- 한국 관광데이터 Open API 기반 런트립 코스 추천
- 한국 기상청 API 연동 실시간 날씨 및 마라톤 대회 정보 공유
- 러닝코스 인근 관광지 추천 (음식점, 문화시설, 행사 등)

<br/>
